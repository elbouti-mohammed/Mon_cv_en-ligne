# 💼 CV en Ligne — Mohamed Elbouti
 
> CV personnel en ligne réalisé dans le cadre du module **Obtimization Web** — DUT Marketing Digital & Expérience Utilisateur, EST Dakhla (2025–2026).
 
---
 
## 📋 Description du projet
 
Ce projet est un **CV interactif et responsive** développé avec **HTML5, CSS3 et Bootstrap 5**. Il présente mon profil professionnel de marketeur digital, mes compétences en communication digitale et réseaux sociaux, mon expérience de stage, mes projets universitaires (WooCommerce, NOVA FITNESS), mon projet de fin d'études (PFE) sur l'IA et le comportement des consommateurs, ainsi qu'un formulaire de contact fonctionnel.
 
Le site adopte un design **sombre et moderne** (fond dark, accents violet/rose) et est entièrement responsive (mobile, tablette, desktop). Il est hébergé sur **GitHub Pages**.
 
---
 
## 🛠️ Technologies utilisées
 
| Technologie | Usage |
|---|---|
| **HTML5** | Structure sémantique du contenu |
| **CSS3** | Styles personnalisés, variables CSS (:root), animations |
| **Bootstrap 5.3** | Grille responsive, composants (cards, progress bars, badges) |
| **Bootstrap Icons 1.11** | Iconographie dans toutes les sections |
| **Google Fonts** | Fraunces (titres) + Plus Jakarta Sans (corps) |
| **JavaScript (Vanilla)** | Formulaire de contact + animation scroll IntersectionObserver |

## 📸 Captures d'écran
 
### 1. Section Hero — En-tête dark avec nom, rôle et coordonnées

### 2. Code VS Code — main.js (JavaScript)
> *Le fichier main.js contient la fonction sendEmail() avec validation regex et l'IntersectionObserver pour les animations scroll.*
 
### 3. Code VS Code — style.css (Variables CSS)
> *Le fichier style.css commence par les variables CSS :root (--violet, --rose, --teal, --bg, --ink…) suivies des styles body utilisant la police Plus Jakarta Sans.*
 
### 4. Code VS Code — index.html (Structure Bootstrap)
